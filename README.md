# E-commerce and Chatbot React project

Working E-commerce and Chatbot webpages following [SuperSimpleDev's React course](https://github.com/SuperSimpleDev/react-course/tree/main)

E-commerce:
- Home page with grid of products that can be added to cart
- Checkout page showing items in cart, shipping options per item, order totals, and option to place order
- Orders page that shows past orders
- Tracking page that emulates shipping status of current/past orders
- Component-ized layout using React
- Not currently hosted anywhere, but here is an [example of how the page looks](https://supersimple.dev/projects/ecommerce/)

Chatbot:
- Chat page that allows sending messages to a simple chatbot ('Hello', roll a dice, flip a coin, time of day)
- Component-ized layout using React
- Not currently hosted anywhere, but here is an [example of how the page looks](https://supersimple.dev/projects/chatbot/)
